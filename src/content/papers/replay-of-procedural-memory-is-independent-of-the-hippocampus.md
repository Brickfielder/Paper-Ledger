## Summary
The authors asked whether sleep-dependent replay of procedural memories requires the hippocampus or can arise within the striatal circuit that supports the skill itself. To test this, they built a multistep mouse task in which animals learned a fixed five-port nose-poke sequence from memory. They first showed that the dorsolateral striatum (DLS) was necessary for learning and executing the sequence, using bilateral DLS lesions and DLS inactivation. They then disrupted offline striatal plasticity by infusing the NMDA receptor antagonist AP5 into the DLS immediately after training, before sleep. This manipulation impaired next-day performance, especially during early learning, indicating that offline DLS processes support consolidation of the skill. To look for a replay-like mechanism, they recorded DLS activity with Neuropixels probes during task performance and subsequent sleep. Rather than using a standard template-matching replay detector, they applied an unsupervised point-process model called PP-Seq to identify latent neural sequences without predefining behavioral or temporal assumptions. The model revealed multiple striatal neural sequences that lined up with distinct phases of the learned action sequence, even though the model had no access to the behavior. These sequences were not limited to task execution, because some also tracked reward-related or other spontaneous behaviors. When the same data were examined during post-task sleep, the awake-identified striatal sequences reappeared offline as replay events. The replay preserved both the order of spikes within a neural sequence and, when multiple sequences co-occurred, the order in which those sequences had appeared during wake behavior. Most importantly, all of these replay features persisted after complete bilateral hippocampal lesions. The central conclusion is that procedural replay can occur in the dorsal striatum independently of the hippocampus, supporting the idea that replay-driven consolidation can operate through parallel memory systems rather than requiring a universal hippocampal orchestrator.
## Why This Matters
This study challenges the common idea that the hippocampus is always the source of sleep replay for memory consolidation. It suggests that skill learning can be reinforced by replay generated directly inside the corticostriatal system, which changes how we think about different kinds of memory being stabilized during sleep.
## Caveats and Limitations
The work is strong mechanistically, but it is still limited to one procedural task in mice, so it is not clear how broadly the result generalizes across other skills or species. The replay detection relied on an unsupervised model, which is a useful strength, but also means the identified sequences depend on modeling choices and would benefit from additional validation with complementary approaches. The hippocampal lesion result is important, yet lesions can have compensatory or circuit-wide effects, so the findings support independence but do not fully rule out more subtle hippocampal contributions in other settings.
## Key Takeaways
- Procedural memory replay was found in dorsal striatum, not just hippocampus.
- The replay was observed during sleep and predicted later performance gains.
- An unsupervised sequence-detection method (PP-Seq) avoided strong preconceptions about replay shape.
- Striatal replay preserved both within-sequence spike order and the order between sequences.
- Complete bilateral hippocampal lesions did not remove the striatal replay features.
